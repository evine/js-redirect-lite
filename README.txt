== Description ==

Redirects to a specific URL using JavaScript

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Edit 'js-redirect-lite.php' on line #30 change the URL value you want to redirect to.
2. Upload `js-redirect` folder to to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Test by going to the homepage. It should redirect to the redirect url set.
5. Go to homepage again. Page will not redirect.
6. Go to your-url/?js_redirect_reset=1 - this will delete the cooke so that you can re-test.
